## Getting Started

1. **Open the website:**
```bash
# Using Python 3
python3 -m http.server 8000

# Or using PHP
php -S localhost:8000

# Or using Node.js (if installed)
npx serve
```

2. **Open Browser:**
Navigate to [http://localhost:8000](http://localhost:8000)
